# 🤖 LOGI - AI LOG EXPLAINER

## TR Türkçe

**Logi**, karmaşık sistem loglarını ve hata mesajlarını analiz etmek için geliştirilmiş akıllı bir asistan uygulamasıdır. Kullanıcının yapıştırdığı log verilerini inceler ve Google Gemini API aracılığıyla çözüm önerileri üretir.

### 🚀 Özellikler

* 📝 Kullanıcıdan log verisi alma
* 🧠 Yapay zeka ile anlık hata analizi
* 💡 Adım adım çözüm önerileri üretme
* 🌙 Modern ve kullanıcı dostu arayüz (Streamlit)
* 🛡️ Güvenli API anahtarı yönetimi (.env)

---

### 🛠️ Kullanılan Teknolojiler

* Python 3
* Streamlit (Arayüz)
* Requests (API işlemleri)
* Google Gemini API
* Python-dotenv (Güvenlik)

---

### ⚙️ Kurulum

git clone [https://github.com/thezeyno/Logi-AI-Log-Explainer.git](https://github.com/thezeyno/Logi-AI-Log-Explainer.git)
cd Logi-AI-Log-Explainer
pip install streamlit requests python-dotenv
streamlit run app.py

---

### 🔑 API Anahtarı
Projenin kök dizininde bir .env dosyası oluşturun:
GEMINI_API_KEY = "YOUR_API_KEY"

### ⚠️ API anahtarınızı GitHub'a yüklemeyin! (.gitignore dosyasını kontrol edin.)

---

### 🎯 Amaç
Bu proje, karmaşık hata loglarını herkesin anlayabileceği basit bir dile dönüştürerek hata ayıklama (debugging) sürecini hızlandırmayı amaçlar.

### GB English
Logi is an intelligent assistant application developed to analyze complex system logs and error messages. It examines user-pasted log data and generates solution suggestions via the Google Gemini API.

---

### 🚀 Features
* 📝 Accepts log input from user
* 🧠 Real-time error analysis with AI
* 💡 Generates step-by-step solution suggestions
* 🌙 Modern and user-friendly interface (Streamlit)
* 🛡️ Secure API key management (.env)

---

### 🛠️ Technologies Used
* Python 3
* Streamlit (UI)
* Requests (API communication)
* Google Gemini API
* Python-dotenv (Security)

---

### ⚙️ Installation
git clone [https://github.com/thezeyno/Logi-AI-Log-Explainer.git](https://github.com/thezeyno/Logi-AI-Log-Explainer.git)
cd Logi-AI-Log-Explainer
pip install streamlit requests python-dotenv
streamlit run app.py

---

### 🔑 API Key
Create a .env file in the project root:
GEMINI_API_KEY = "YOUR_API_KEY"

### ⚠️ Do not upload your API key to GitHub! (Check your .gitignore file.)

---

### 🎯 Purpose
This project aims to speed up the debugging process by converting complex error logs into simple language that everyone can understand.
